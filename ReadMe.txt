####################
Utilisation
####################
-1 lancer interface.py
-2 Insert -> puis selectionner un CV (le votre peut-etre) en docx.
-3 Clicker sur le CV ajouté (devrais etre le path) et appuyer sur Select Text
-3 Les CVs sont jugés sur une base de bien intégrer une équipe d'adjointe administrative,
ne soyer pas surpris si le pourcentage n'équivaux pas compétences dans d'autres domaines.
-4 Appuyer sur Apply
-5 Le Résultat s'affiche sur la barre au bas de l'écran suivi d'un pourcentage.

####################
Evaluation
####################
-1 interface.py contient l'interface
-2 MainOperation.py contient le Naive Bayes Classifier et les méthode de "purification" des données
